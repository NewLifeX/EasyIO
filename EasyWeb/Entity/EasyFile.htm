文件仓库（FileStorage）

名称 显示名 类型 长度 精度 主键 允许空 备注
Id 编号 Int32 AI N
Name 名称 String 50 UQ
Enable 启用 Boolean N
HomeDirectory 主目录 String 50
Size 大小 Int64 N 总大小
Status 状态 Int32 N
Level 深度 Int32 N 最大搜索目录深度
Period 同步周期 Int32 N 默认60秒
Pattern 匹配规则 String 50 仅搜索匹配的文件，支持*，多个规则逗号隔开
RedirectMode 原始跳转 Int32 N 跳转到原始地址
VipUrl VIP地址 String 50 CDN域名地址，加速下载
VipKey VIP密钥 String 50 CDN的URL验证密钥
LastScan 最后扫描 DateTime 记录最后一次扫描时间
CreateUserId 创建者 Int32 N
CreateTime 创建时间 DateTime
CreateIP 创建地址 String 50
UpdateUserId 更新者 Int32 N
UpdateTime 更新时间 DateTime
UpdateIP 更新地址 String 50
Remark 备注 String 500


文件清单（FileEntry）

名称 显示名 类型 长度 精度 主键 允许空 备注
Id 编号 Int32 AI N
StorageId 仓库 Int32 N
SourceId 来源 Int32 N
Name 名称 String 250
Title 标题 String 50
FullName 全路径 String 250 文件保存在本地的相对路径
Path 路径 String 250 用于Url的相对路径
Enable 启用 Boolean N
ParentId 上级目录 Int32 N 0表示顶级
IsDirectory 目录 Boolean N 当前清单是目录而不是文件
Size 大小 Int64 N 文件大小
Version 版本 Int32 N 用于排序
Tag 标签 String 50 类型标签
LastWrite 最后修改 DateTime
LastAccess 最后访问 DateTime
LastScan 最后扫描 DateTime 最后一次扫描访问的时间
Hash 哈希 String 600 MD5哈希或SHA512
RawUrl 原始地址 String 500 文件的原始地址，如果文件在本地不存在时，跳转原始地址
RedirectMode 原始跳转 Int32 N 跳转到原始地址
LinkTarget 链接目标 String 250 链接到目标文件，支持*和!*匹配目标目录的最新匹配文件
LinkRedirect 链接跳转 Boolean N 跳转到目标文件
Times 次数 Int32 N 下载次数
LastDownload 最后下载 DateTime
TraceId 链路 String 200 链路追踪
CreateUserId 创建者 Int32 N
CreateTime 创建时间 DateTime
CreateIP 创建地址 String 50
UpdateUserId 更新者 Int32 N
UpdateTime 更新时间 DateTime
UpdateIP 更新地址 String 50
Remark 备注 String 500

Enable 启用 Boolean N

Whites 白名单 String 50 仅搜索匹配的文件，支持*，多个规则逗号隔开
Blacks 黑名单 String 50 仅搜索匹配以外的文件，支持*，多个规则逗号隔开
LastScan 最后扫描 DateTime 记录最后一次扫描时间
CreateUserId 创建者 Int32 N
CreateTime 创建时间 DateTime
CreateIP 创建地址 String 50
UpdateUserId 更新者 Int32 N
UpdateTime 更新时间 DateTime
UpdateIP 更新地址 String 50
Remark 备注 String 500
